# KSU Specialization Calculator

A modern web application that helps King Saud University students calculate:

- Cumulative GPA
- Specialization Score
- Eligible Majors
- Difference from each major cutoff

## Features

- Multi-Step Interface
- GPA Calculation
- Specialization Score Calculation
- Dark / Light Mode
- Responsive Design
- Pure HTML, CSS, and JavaScript

## Formula

Specialization Score = (GPA × 10) + (Qudrat × 25%) + (Tahsili × 25%)

## Technologies

- HTML5
- CSS3
- JavaScript
