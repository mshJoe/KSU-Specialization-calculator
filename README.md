## ✨ Key Features

*   **🧮 Precise Calculations:** Accurately computes Cumulative GPA and combines it with Qudrat (25%) and Tahsili (25%) to generate the final Specialization Score (out of 100).
*   **🎯 Smart Major Matching:** Instantly cross-references the user's score with KSU's historical acceptance rates, categorizing majors into "Eligible" and "Out of Reach" (including the exact point difference needed).
*   **🌍 Bilingual Interface:** Full support for both English and Arabic languages with dynamic RTL/LTR layout switching.
*   **🌓 Dark/Light Mode:** A sleek theme toggler that adapts to user preference, complete with dynamic KSU logo switching for maximum visual harmony.
*   **📱 Fully Responsive:** Clean, intuitive step-by-step UI optimized for desktop, tablet, and mobile devices.
*   **📊 Integrated Analytics:** Empowered by Google Analytics to track user engagement and real-time site performance.

## 🚀 How to Use

1.  **Step 1 (Courses):** Enter the number of courses taken during the first year.
2.  **Step 2 (Grades):** Input the credit hours and the achieved grade for each course.
3.  **Step 3 (Test Scores):** Enter your national aptitude test scores (Qudrat & Tahsili).
4.  **Results:** View a comprehensive breakdown of your GPA, total specialization score, your academic performance rank, and a detailed list of college availability.

## 🛠️ Local Setup & Installation

Since this project is built using vanilla web technologies, no complex build tools or package managers are required.

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/mshJoe/KSU-Specialization-calculator.git](https://github.com/mshJoe/KSU-Specialization-calculator.git)
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd KSU-Specialization-calculator
    ```
3.  **Run the application:**
    Simply open the `index.html` file in your preferred web browser.

## 📁 File Structure

*   `index.html` — The main structure and content of the application.
*   `style.css` — Custom styling, CSS variables, responsive design, and theme logic.
*   `script.js` — Application logic, state management, UI interactions, and localization strings.
